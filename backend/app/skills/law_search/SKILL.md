# 法規搜尋 Skill

## 概述

本 skill 提供台灣全國法規資料庫的搜尋能力，涵蓋 1,343 部法律 + 10,409 部命令，共 11,752 部現行法規。

## 搜尋流程

1. **判斷所屬類別**：根據下方法規分類大綱，判斷要搜的法規屬於哪個部會/目別
2. **限縮搜尋**：用 `search_law(query, category_prefix)` 在該類別內搜尋
3. **查條文**：用 `get_article(law_name, article_no)` 取得具體條文
4. **驗證引用**：用 `verify_citation(citation)` 確認「○○法第○條」是否正確

## 工具

### search_law(query, category_prefix?, top_k?)
搜尋法規名稱。**建議帶 category_prefix 限縮範圍**，避免搜到不相關的法。
- `query`: 關鍵字（如「勞工保險」「公司」「食品安全」）
- `category_prefix`: 類別前綴（如「行政＞勞動部」「行政＞經濟部＞商業目」）。參考下方大綱。
- 例：`search_law("公司", "行政＞經濟部＞商業目")` → 找到公司法，不會找到金融控股公司法

### get_article(law_name, article_no?)
取得法規條文。
- 不帶 article_no → 回傳前 5 條預覽
- `article_no`: 如「第20條」「20」

### verify_citation(citation)
驗證引用格式。
- 輸入：「勞工保險條例第20條」「行政程序法第154條第1項」
- 回傳：是否有效、條文內容

## 法規分類大綱

以下是現行法規的部會/目別分類，括號內為該目別的法規數量。搜尋前先判斷所屬類別，用 category_prefix 限縮搜尋範圍。

完整大綱請讀取 CATEGORIES.txt。以下為常用部會摘要：

### 常用部會
- **內政部**(888): 警政目, 國土管理目, 地政目, 戶政目, 消防目, 移民目, 役政目
- **財政部**(392): 賦稅目, 國庫目, 關務目
- **教育部**(506): 國民及學前教育目, 高等教育目, 終身教育目, 學生事務及特殊教育目
- **法務部**(338): 檢察目, 矯正目, 法律事務目, 廉政目
- **經濟部**(682): 商業目, 能源目, 水利目, 產業目, 智慧財產權目, 標準檢驗目
- **交通部**(624): 公路目, 航空目, 航政目, 觀光目, 鐵路目, 郵政目
- **衛生福利部**(716): 醫事目, 食品藥物管理目, 社會及家庭目, 國民健康目, 疾病管制目
- **勞動部**(278): 職業安全衛生目, 就業服務目, 勞動保險目, 勞動條件及就業平等目
- **農業部**(611): 動植物防檢疫目, 漁業目, 農糧目, 農民輔導目, 農業金融目
- **環境部**(347): 大氣環境目, 化學物質管理目, 資源循環目, 水質保護目
- **金融監督管理委員會**(389): 銀行目, 證券暨期貨管理目, 保險目
- **國家科學及技術委員會**(73): 科學技術目, 園區目

### 搜尋技巧
- 搜「勞基法」→ category_prefix=「行政＞勞動部＞勞動條件及就業平等目」
- 搜「所得稅」→ category_prefix=「行政＞財政部＞賦稅目」
- 搜「食品」→ category_prefix=「行政＞衛生福利部＞食品藥物管理目」
- 搜「建築」→ category_prefix=「行政＞內政部＞國土管理目」
- 不確定類別時，不帶 category_prefix 搜全部，但結果可能不精確
